How to Use

Follow these simple steps to install Microsoft Office

1

Download Both Files

Download both Install-Office-SAFE.bat and Install-Office.ps1 to the same folder (like Downloads).

2

Double-Click the BAT File

Double-click Install-Office-SAFE.bat to start the installer. This handles all the technical stuff automatically.

3

Accept Admin Permission

Click "Yes" when Windows asks for administrator permission. This is required for Office installation.

4

Follow the Prompts

The installer will guide you through simple questions about what you want to install. Just pick what sounds right!

5

Wait for Installation

Office will download and install automatically (10-30 minutes). Perfect time for a coffee break!

6

Start Using Office

Once complete, find Word, Excel, PowerPoint, and other Office apps in your Start Menu. Enjoy!

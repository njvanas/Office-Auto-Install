Office Auto Installer

Easy Microsoft Office installation for everyone. No ZIP required — admin PowerShell: irm "…/office.ps1" | iex (same idea as WinUtil), or download the classic packages below.

Usage (WinUtil-style)

Same launch model as Chris Titus Tech’s WinUtil: Administrator PowerShell, then irm … | iex. This project uses office.ps1 as the single bootstrap entry (like WinUtil’s short URL).

Run as Administrator

  1. Start menu: Right-click the Start button → Terminal (Admin) or Windows PowerShell (Admin).
  2. Search: Press Win, type PowerShell or Terminal, then Ctrl+Shift+Enter or right-click → Run as administrator.

Launch command

Stable (recommended) — GitHub Pages, shorter URL

irm "https://njvanas.github.io/Office-Auto-Install/office.ps1" | iex

Stable — raw GitHub (fallback if Pages is blocked)

irm "https://raw.githubusercontent.com/njvanas/Office-Auto-Install/main/office.ps1" | iex

Dev branch payloads (bootstrap still from main/Pages; sets branch for Install-*.ps1)

$env:OFFICE_AUTO_INSTALL_BRANCH = "dev"
irm "https://njvanas.github.io/Office-Auto-Install/office.ps1" | iex

Console wizard (instead of GUI)

$env:OFFICE_AUTO_INSTALL_USE_CONSOLE = "1"
irm "https://njvanas.github.io/Office-Auto-Install/office.ps1" | iex

Forks: set $env:OFFICE_AUTO_INSTALL_REPO = "yourname/Office-Auto-Install" before irm. Custom domain (like christitus.com/win): point it at this repo’s GitHub Pages site or raw office.ps1. Only run irm … | iex from sources you trust.

Renamed from Launch-Office.ps1 → office.ps1 for a single WinUtil-like entry point.

How to Use

Fastest path: Run online with office.ps1 (WinUtil-style). Below are folder-based steps if you already downloaded the ZIP or individual files.

GUI Version (Recommended) ⭐

Modern Windows 11 Fluent Design interface

1

Download Files

Download both Install-Office-GUI-SAFE.bat and Install-Office-GUI-WPF.ps1 to the same folder.

2

Run the Launcher

Double-click Install-Office-GUI-SAFE.bat and click "Yes" when Windows asks for administrator permission.

3

Use the GUI

The beautiful GUI window will open with all options pre-configured. Click "Install Office" or customize settings first.

4

Monitor Progress

Watch real-time progress with visual feedback as Office downloads and installs.

5

Wait for Installation

Office will download and install automatically (10-30 minutes). Don't close the window!

6

Start Using Office

Find Word, Excel, PowerPoint, and other Office apps in your Start Menu.

Console Version

Step-by-step text-based interface

1

Download Files

Download both Install-Office-SAFE.bat and Install-Office.ps1 to the same folder.

2

Run the Launcher

Double-click Install-Office-SAFE.bat and click "Yes" when Windows asks for administrator permission.

3

Follow Prompts

The installer will guide you through simple questions about what you want to install.

4

Confirm Choices

Review your selections before the download begins.

5

Wait for Installation

Office will download and install automatically (10-30 minutes). Keep the window open!

6

Start Using Office

Find Word, Excel, PowerPoint, and other Office apps in your Start Menu.

⚠️ Important Notes

  • • Both files must be in the same folder for this to work
  • • You need administrator rights - Windows will ask for permission
  • • Stay connected to the internet - Office downloads during installation
  • • Don't close the window during installation (10-30 minutes)
